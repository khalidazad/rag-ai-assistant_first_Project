# 🧠 RAG AI Assistant (PDF-Based Retrieval Augmented Generation)

A full-stack Retrieval-Augmented Generation (RAG) application that allows users to upload a PDF and chat with it using semantic search and LLM-powered responses.

Built using:
- Node.js (Express)
- React (Vite)
- OpenAI-compatible API (OpenRouter)
- Vector embeddings
- Cosine similarity search

---

## 🚀 Features

✅ PDF upload & text extraction  
✅ Text chunking with overlap  
✅ Embedding generation  
✅ Cosine similarity search  
✅ Top-K retrieval  
✅ Context grounding (anti-hallucination)  
✅ Chat-style UI  
✅ In-memory vector store (MVP)
